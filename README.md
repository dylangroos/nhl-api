# nhl-api
api with endpoints to return nhl game data

# Under Construction...

`Node v17.4.0 via n or nvm`

# Installation
Clone repo

`git clone https://github.com/dylangroos/nhl-api.git`

Install packages

`cd nhl-api`

`npm install`

Run server

`node server.js`

Access root on localhost

goto `https://localhost:8080/api`
